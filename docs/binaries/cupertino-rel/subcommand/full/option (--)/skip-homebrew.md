# --skip-homebrew

Skip Homebrew formula update.

## Synopsis

```bash
cupertino-rel full <version-or-type> --skip-homebrew
```

## Description

Skip updating the Homebrew formula. The formula in `mihaelamj/homebrew-tap` won't be updated.

## Use Case

- Pre-release versions
- When Homebrew tap access is unavailable
- Testing release workflow
