# Settings View

Configure Cupertino options.

## Description

View and modify Cupertino configuration settings.

## Settings

| Setting | Description |
|---------|-------------|
| Output Directory | Base directory for downloaded documentation |
| Search DB Path | Path to the search database |
| Samples DB Path | Path to the samples database |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `↑` / `k` | Move selection up |
| `↓` / `j` | Move selection down |
| `Enter` | Edit setting |
| `h` / `Esc` | Return to home (cancel edit) |
| `q` | Quit |

## Editing

Press `Enter` on a setting to edit. Type the new value and press `Enter` to save, or `Esc` to cancel.
