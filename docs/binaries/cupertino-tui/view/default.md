# cupertino-tui views

Available views in the Terminal UI.

## Views

| View | Description |
|------|-------------|
| [home](home.md) | Main menu with quick stats and navigation |
| [packages](packages.md) | Browse and search Swift packages |
| [library](library.md) | View artifact collections |
| [archive](archive.md) | Browse classic Apple programming guides |
| [settings](settings.md) | Configure Cupertino options |

## Navigation

- Use number keys `1-4` for quick selection from home
- Use `h` or `Esc` to return to home
- Use `q` to quit
