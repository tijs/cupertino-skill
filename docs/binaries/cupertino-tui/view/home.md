# Home View

Main menu and quick stats.

## Description

The home view is the entry point for the TUI. It displays quick stats about your documentation and provides navigation to other views.

## Quick Stats

- **Packages** - Number of selected packages
- **Downloaded** - Number of downloaded packages
- **Total Size** - Disk space used

## Menu Items

| Key | Item | Description |
|-----|------|-------------|
| `1` | Packages | Browse Swift packages |
| `2` | Library | View artifact collections |
| `3` | Archive | Browse Apple guides |
| `4` | Settings | Configure options |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `↑` / `k` | Move selection up |
| `↓` / `j` | Move selection down |
| `Enter` | Select item |
| `1-4` | Quick select |
| `q` | Quit |
