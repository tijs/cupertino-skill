# Archive View

Browse classic Apple programming guides.

## Description

Browse 46+ legacy Apple programming guides from the Apple Archive. These are pre-2016 guides covering foundational topics.

## Available Guides

- Core Animation Programming Guide
- Quartz 2D Programming Guide
- Key-Value Coding Programming Guide
- Key-Value Observing Programming Guide
- Threading Programming Guide
- Core Text Programming Guide
- And more...

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `↑` / `k` | Move selection up |
| `↓` / `j` | Move selection down |
| `/` | Start search |
| `Enter` | View guide |
| `h` / `Esc` | Return to home |
| `q` | Quit |

## Search

Press `/` to filter guides by name.
