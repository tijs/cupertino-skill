# Library View

View artifact collections.

## Description

Browse the downloaded documentation artifacts organized by collection type.

## Collections

- **Apple Docs** - Apple Developer Documentation
- **Swift Evolution** - Swift Evolution proposals
- **Swift.org** - Swift.org documentation
- **HIG** - Human Interface Guidelines
- **Sample Code** - Apple sample projects

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `↑` / `k` | Move selection up |
| `↓` / `j` | Move selection down |
| `Enter` | View collection |
| `h` / `Esc` | Return to home |
| `q` | Quit |
